Este archivo se divide en dos mains diferentes.

MainAsteroides
Este se encarga de manejar el programa de asteroides, donde tienes que ingresar la cantidad de asteroides. Posteriormente escribes el tamaño que tienen negativos o positivos, pero sin
incluir al 0, despues llamada a SimuAsteroides, donde se encargar de verificar los choques de los asteroides, y al final devuelve los asteroides restantes.

Etiquetas
Recibe etiquetas de forma HTML, y empieza a verificar si esta balanceada.
Tiene que empezar con body, y cada etiqueta de apertura tiene una de cierre.
Al final si la pila donde se guardan las etiquetas es vacia, no quedaron elementos de cierre sin apertura y empezo con body entonces esta balanceada, caso contrario no lo esta
